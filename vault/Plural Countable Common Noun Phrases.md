Plural countable common noun phrases can be collective:

```
the cards
some cards
3 cards
three cards
```

or distributive:

```
each of the cards
each of some cards
each of 3 cards
each of three cards
```

Integers up to 12 used as determiner as in `12 cats` can be written in characters, i.e. `twelve cats`. Larger integers must be written in digits.
